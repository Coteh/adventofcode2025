Advent of Code 2025 solutions. Trying [Zig](https://ziglang.org/) this year.

## Instructions

To compile, and then run:

```sh
zig build-exe 01/main.zig

./main < 01/input
```

To compile and run in one step:

```sh
zig run 01/main.zig < 01/input
```
